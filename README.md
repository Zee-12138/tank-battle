# 坦克大战 (Tank Battle) 🎮

经典红白机风格坦克大战网页游戏，纯 HTML/CSS/JavaScript 实现，无需任何依赖，打开即玩！

## 🕹️ 游戏特色

- **经典玩法**：致敬经典 FC 坦克大战，保护基地，消灭所有敌方坦克
- **多个关卡**：3 个精心设计的地图，每关敌人递增
- **多种敌人**：普通坦克、快速坦克、重型坦克、装甲坦克
- **精美画面**：Canvas 绘制，粒子爆炸特效，坦克细节丰富
- **音效系统**：Web Audio API 合成音效（射击、爆炸等）
- **移动端支持**：触摸滑动移动 + 点击射击
- **键盘操作**：方向键移动，空格射击，P 暂停

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` 或 `W` `A` `S` `D` | 移动坦克 |
| `空格` 或 `J` | 发射子弹 |
| `P` | 暂停/继续 |
| `Enter` | 开始游戏 |

## 🏗️ 地图元素

- 🧱 **砖墙**：可被子弹摧毁
- 🔩 **钢铁墙**：不可摧毁
- 🌊 **水域**：坦克无法通过，子弹可穿过
- 🌳 **树林**：遮挡视线，坦克和子弹可穿过
- 🦅 **基地**：保护好它！被摧毁则游戏结束

## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html`

或者直接访问 GitHub Pages 在线游玩！
